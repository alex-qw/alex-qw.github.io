Mindbody Email Link Creator

  1. [REQUIRED] Enter a web URL to send desktop traffic
  2. [REQUIRED] Select web-only if link should avoid launching app
  3. [REQUIRED] Add analytics values
  4. [OPTIONAL] Add a $deeplink_path to open a specific app screen (use Leehey's rubric)
  5. [OPTIONAL] Set $android_url and $ios_url for users without the app (currently defaults to the app store, but will default to the desktop url if removed entirely)
  6. [OPTIONAL] To create a link that always sends users to the web, check the "Always send users to web" checkbox and remove $android_url and $ios_url by clicking "Remove Query Param"
  7. Click Create Email Link. Link data can be edited after their built, by changing the values directly on the url.
